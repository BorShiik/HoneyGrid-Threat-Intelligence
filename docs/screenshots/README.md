# Screenshots

Drop the dashboard captures here with **exactly these filenames** — the main
[`README.md`](../../README.md) references them and they render automatically
once the files exist.

| Filename | Page to capture |
|---|---|
| `dashboard.png` | Main dashboard (hero — KPIs + feed + 3D globe) |
| `threat-map.png` | Live Attack Map (global arcs) |
| `live-feed.png` | Live Feed (expanded event: session dump + threat intel) |
| `threat-actors.png` | Threat Actors (AI actor dossiers) |
| `session-replay.png` | Session Replay (TTY playback) |
| `credentials.png` | Credentials (top usernames/passwords) |
| `ioc-stix.png` | IoC / STIX feed |
| `ai-integrations.png` | AI Integrations |
| `sdn-monitoring.png` | SDN Monitoring |

**Tips**
- Use a consistent viewport (e.g. 1600×900) and the dark theme for a uniform look.
- PNG preferred; keep each file under ~1 MB (compress if needed).
- For a crisp hero image, capture the dashboard with live data flowing.
